# AI Usage Disclosure

**Tool Used**: Gemini (Google AI)

## How I Used AI in This Project

I used Gemini mainly as a personal tutor and to help me organize my thoughts better. I wanted to make sure my presentation and documentation were clear and professional. Here is exactly what I did with it:

1.  **Grammar and Phrasing**: Since English is not my first language, I asked Gemini to check my README and project descriptions. It helped me fix some grammar mistakes and suggest better words so it sounds more natural.
2.  **Brainstorming Data**: I had a hard time thinking of what kind of data to put in my database. I asked Gemini for ideas for a "Retail Store" database schema. It suggested some table names like `customers`, `orders`, and `products`. I then wrote the SQL myself based on these ideas.
3.  **Troubleshooting Docker**: I got a few errors when I first tried to run `docker-compose`. I pasted the error message into Gemini, and it explained that I needed to use `docker compose` (without the dash) for the version I have. It helped me understand the error instead of just guessing.
4.  **Learning DBeaver Features**: I wasn't sure how "Execution Plan" or "ER Diagrams" worked in DBeaver. I asked Gemini for a simple explanation, and it helped me understand why they are useful for Data Engineering.

## What I Did Myself

It was important for me to learn the tools properly, so I did the heavy lifting:
-   **SQL Coding**: I wrote all the table structures and the queries for the demo.
-   **Presentation**: I designed all the slides and recorded the demo video myself.
-   **Environment Setup**: I installed Docker and DBeaver and made sure everything connected correctly on my Mac.

I used Gemini to help me learn and polish my work, but the actual logic and the presentation are my own work.
